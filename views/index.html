Javel

A Javascript router with Laravel-like syntax.

Github

Installation

For now, just include the javel.js and routes.js files to the footer of the html. Also make sure you have a div (or something) with the id of 'view', as that's where the content is loaded.

Making a Route


Route.get('myroute', function() {
    return 'This will simply be echoed in the browser';
});

Making a Route with a View

Put views in the 'views' folder, with the filename as the name of the view with an .html extension. Make sure to pass 'app' to the callback and View.make.


Route.get('myview', function(app) {
    return View.make('myview', app);
});
